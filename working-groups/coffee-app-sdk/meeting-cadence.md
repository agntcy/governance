# WG Meeting Cadence

- **Biweekly Community Call** (Video). Duration: 45 min.  
- **Standing Agenda**:  
  1. Status updates  
  2. Blockers / escalations  
  3. New proposals / RFCs  
  4. Action items  

- **Async updates**: Slack `#coffee-agntcy-app-sdk`.  
- **Minutes**: Stored in GitHub under `meetings/` folder.
